lptrends тестовое задание.

Этот проект представляет собой таблицу пользователей и их электронных почтовых ящиков с возможностью добавления и удаления записей талицы, а также валидацией вводимых значений.

Произвести сборку проекта можно командой npm run build.
Также предусмотрена возможность запуска проекта через webpack-dev-server, для этого необходимо использовать команду npm run start:dev.
